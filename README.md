## Hi, I'm Ethan

Software Engineer at Invista / Koch Industries. I studied Computer Science at Wichita State University and spend my personal time building things I actually want to use.

---

### Projects

**[Portfolio](https://github.com/EthanMW2000/portfolio)**
Personal site built with Next.js, React, and MUI. Includes a photography gallery backed by S3 and CloudFront, with a private admin upload page protected by GitHub OAuth.

**[Image Processor](https://github.com/EthanMW2000/image-processor)**
AWS Lambda that watches an S3 bucket and automatically resizes and converts uploaded photos to WebP — one version for gallery view, one for thumbnails. Deployed with Terraform via GitHub Actions.

**[Spectrum Analyzer](https://github.com/EthanMW2000/spectrum-analyzer)**
Raspberry Pi-powered real-time 32-band audio visualizer with a vintage phosphor CRT aesthetic. Identifies songs via Shazam and displays album art. Runs headless as a systemd service next to my HiFi setup.
